# One-to-One Chat Application

A **full-stack, real-time one-to-one chat application** enabling secure and seamless messaging between users. The project includes both **frontend and backend**, demonstrating modern web development practices and real-time communication.

---

## Features
- Real-time one-to-one messaging using **Socket.io**  
- **User authentication** with signup and login  
- **Frontend:** React.js with **Redux Toolkit** for state management and **DaisyUI** for UI components  
- **Backend:** Node.js and Express.js with secure APIs  
- Responsive and intuitive user interface  
- Easily extendable for future enhancements  

---

## Technology Stack
- **Frontend:** React.js, Redux Toolkit, DaisyUI, JavaScript  
- **Backend:** Node.js, Express.js, Socket.io  
- **Database:** MongoDB / MySQL  
- **Authentication:** bcrypt.js  

---

This application demonstrates **end-to-end full-stack development**, real-time messaging, state management, and secure user handling, making it suitable for **company projects and professional portfolios**.
