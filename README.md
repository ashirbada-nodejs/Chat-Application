# One-to-One Chat Application

A **full-stack, real-time one-to-one chat application** enabling secure and seamless messaging between users. The project includes both **frontend and backend** implementations, showcasing modern web development practices.

---

## Features
- Real-time one-to-one messaging using **Socket.io**  
- **User authentication** with signup and login  
- Responsive and intuitive **frontend interface**  
- Secure and efficient **backend API** built with **Node.js** and **Express.js**  
- Easily extendable for future enhancements  

---

## Technology Stack
- **Frontend:** HTML, CSS, JavaScript (or React if used)  
- **Backend:** Node.js, Express.js, Socket.io  
- **Database:** MongoDB / MySQL  
- **Authentication:** bcrypt.js  

---

This application demonstrates **end-to-end full-stack development**, real-time communication, and secure user management, making it suitable for professional portfolios and company projects.
